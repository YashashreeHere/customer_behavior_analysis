# Customer Shopping Behavior Analysis

## Overview
This project analyzes customer shopping behavior using retail transaction data. Using Python, SQL, and Power BI, the project uncovers insights into customer preferences, purchasing patterns, subscriptions, discounts, and revenue trends.

## Tools Used
- Python (Pandas)
- MySQL
- Power BI
- GitHub

## Project Workflow
1. Data Cleaning & Feature Engineering (Python)
2. Business Analysis (SQL)
3. Interactive Dashboard Creation (Power BI)
4. Reporting & Recommendations

## Key Analyses
- Revenue by Gender
- Top-Rated Products
- Customer Segmentation
- Subscription vs Non-Subscription Analysis
- Revenue by Age Group
- Discount Impact Analysis
- Top Products by Category

## Dashboard Highlights
- Customer Count
- Average Purchase Amount
- Average Rating
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Subscription Analysis

## Key Insights
- Young adults generated the highest revenue.
- Loyal customers formed the largest customer segment.
- Some products showed high dependence on discounts.
- Top-rated products can be prioritized in marketing campaigns.

## Business Recommendations
- Improve subscription benefits.
- Reward loyal customers.
- Optimize discount strategies.
- Focus marketing on high-value customer segments.
